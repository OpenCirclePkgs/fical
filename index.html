Simple iCal filtering

Filter unwanted iCal events.

On the right side enter the calendar subscription URL, then enter which words are allowed to be in the event names and optionally which ones should be blocked, and subscribe the newly generated URL in your calendar app.

Use comma to separate multiple filters
Use comma to separate multiple filters
If the button does not work you can select the new URL and copy it manually

How does it work?

On every request the remote iCal file is fetched directly from the remote server and parsed, after that the calendar is returned as-is with only events that match the allowed keywords and do not contain any of the blocked words. All the data about the request is stored in the URL and is never stored on the server
